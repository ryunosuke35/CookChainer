
# Cook Chainer

料理が上手くなりたいけど、なかなか継続できない。そんな経験ありませんか？  
CookChainerは、ゲーム感覚で楽しく料理を継続できるようになるアプリケーションです。

ポートフォリオURL:http://35.76.196.103/

## 開発言語

- ruby2.6.5
- Rails5.2.5

## 実行手順

```
$ git clone git@github.com:ryunosuke35/CookChainer.git
$ cd CookChainer
$ bundle install
$ rails db:create db:migrate
$ rails s
```
